1. Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages.
It uses tags to define elements such as text, images, tables, links, and formatting styles.

HTML documents are static in nature and are interpreted by web browsers to display content on the screen.

2. Basic Structure of an HTML Document

Every HTML document follows a predefined structure:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    Page Content
</body>
</html>

3. <style> Tag (Internal CSS)

The <style> tag is used to define CSS rules within the HTML document.

Purpose:

Improves visual appearance

Controls layout, colors, fonts, and spacing

Example:
<style>
  body {
    font-family: Arial;
    background-color: #f4f6f8;
  }
</style>

Benefits:

Keeps design consistent

Makes the page more readable

No external CSS file required
